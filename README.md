Book Explorer Web App

 Overview
Book Explorer is a responsive web application that allows users to search for books using the Open Library API and save their favorite books.

 Features
- Search books by title
- Display book covers and author names
- Add and remove favorites
- Favorites stored in localStorage
- Responsive design using Tailwind CSS
- Async/await API integration
- Git branching and pull requests workflow

 Technologies Used
- HTML
- Tailwind CSS
- JavaScript (ES6 Modules)
- Open Library API
- Git & GitHub

 Project Structure
- index.html
- main.js
- fetchBooks.js
- favorites.js

 How to Run Locally
1. Clone the repository
2. Open the project folder
3. Run using Live Server
4. Search and explore books
